# Templates

A collection of templates for various projects.

## Get Started

Create a new project from template of your choice using the following command.

```shell
npm create i
# or
pnpm create i
```

## Templates List

Use pnpm by default.

- [react-ts-vite-unocss](./templates/react-ts-vite-unocss)
